ISyE 312 - Data Management and Analysis for Industrial Engineers

Fundamentals of data management and analysis. Formulating and solving real industrial engineering problems with appropriate data managing and modeling strategies. Fundamental industrial database management strategies, data preprocessing, visualization and modeling techniques; industrial database management and analysis techniques using leading programming software (MySQL and R).]

ISyE 312 is a core industrial engineering course offered at UW-Madison. The class focuses on data management and analysis. It teaches students how to approach real-world problems using systematic, quantitative methods.

Tools and Techniques Used

  • MySQL
  • R Studio
  • Linear Regression
