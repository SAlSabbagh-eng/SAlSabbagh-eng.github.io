LIS 351 - Introduction to Digital Information

Students will explore information access, information representation, usability and information policy issues, and increase their understanding of the social impacts and social shaping of information infrastructures.

LIS 351 is an exciting course at UW-Madison where I’m learning about different technologies and policy issues associated with digital information. In the first major project we learned the fundamentals of web design and development. The class covers how to use HTML, CSS, and accessibility principles to build modern and user-friendly websites.

One of my favorite parts of this class is experimenting with different design ideas and seeing them come to life through code. I’ve learned how to use HTML, CSS, and responsive layouts to improve usability and design quality.

Key Design Concepts

Accessibility
Ensuring all users can navigate and read website content easily
Responsive Design
Creating layouts that adapt to any screen size
Visual Hierarchy
Using headings, spacing, and color to guide user attention

What I’ve Learned while doing the first major project

  • Skill 1: Writing clean HTML structure
  • Skill 2: Styling pages with CSS
  • Skill 3: Validating accessibility and code
